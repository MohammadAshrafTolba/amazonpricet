# amazonpricet
amazonpricet is prices tracking services where the user is prompted via email if the product price ever fall down.
## Dependencies
- requests
- bs4
- sockets
- _thread
- pickles
## Installation
```bash
pip install git+https://github.com/MohammadAshrafTolba/amazonpricet
```
